# Student Performance Analysis using SQL

This project focuses on analyzing student performance data using SQL queries.

## 📊 Project Overview
The dataset contains information about students, including their marks, subjects, and cities. The goal is to extract useful insights using SQL.

## 🔍 Key Analysis Performed
- Calculated average marks for each subject
- Identified students scoring above a certain threshold
- Found top-performing students
- Counted number of students by city

## 🛠 Tools Used
- MySQL Workbench
- SQL

## 📁 Files Included
- student_analysis.sql (contains all SQL queries)

## 📌 Purpose
This project demonstrates basic SQL skills for data analysis and query writing.
